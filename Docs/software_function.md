# This file list the BLIKVM software fuction 
## Features

* Video capture (HDMI/DVI/VGA)
* Keyboard forwarding
* Mouse forwarding
* ATX
* Fan Control
* Fullscreen mode
* Paste text from clipboard
* Networking （VPN）
* Mass Storage Drive (emulate a CD-ROM or Flash Drive)
* Multiport KVM over IP
* Show System info (such as temp)
